    @ac-essentials/misc-util

    Miscellaneous utilities for JavaScript and TypeScript

    This package provides a collection of utility functions and types that can be used in various JavaScript and TypeScript projects. It includes functions for string manipulation, number formatting, object merging, timer management, and more.

    Installation

    You can install the package via npm:

    npm install @ac-essentials/misc-util

    Or via yarn:

    yarn add @ac-essentials/misc-util

    Utility Summary

    Here's a quick overview of the main utilities available in this package (see source for full details):

    Async & Concurrency

    • file-lock: Cross-process file locking utility for async code.
    • mutex: Mutual exclusion lock for synchronizing async operations.
    • semaphore: Counting semaphore for limiting concurrency.
    • signal: Async signaling primitive for event notification.
    • subscribable: Simple event subscription and notification utility.
    • udp-bind-lock: Lock using UDP port binding for cross-process coordination.
    • wait-notifiable: Wait for a condition to be notified asynchronously.
    • subscribable-event: Type-safe event emitter for async code.

    Data Structures

    • binary-heap: Binary heap data structure for efficient priority queue operations.
    • deque: Double-ended queue supporting fast push/pop at both ends.
    • doubly-linked-list: Doubly linked list implementation.
    • linked-list: Singly linked list implementation.
    • native-array: Wrapper for native arrays with extra utility methods.
    • priority-queue: Priority queue built on a binary heap.
    • queue: FIFO queue data structure.
    • stack: LIFO stack data structure.

    ECMAScript Utilities

    • aggregate-error: Helpers for working with AggregateError and error-like objects.
    • capture-stack-trace: Capture and parse stack traces from errors.
    • error: Helpers for working with Error and error-like objects.
    • error-stack: Parse and format error stack traces.
    • format-error: Format errors and their causes for readable output.
    • traverse-error: Traverse error chains and aggregates.
    • unimplemented-error: Error class for unimplemented features.
    • unsupported-error: Error class for unsupported features or environments.
    • abortable: Wrap a function to make it abortable via AbortSignal.
    • debounce-queue: Debounce async functions so only one runs at a time.
    • no-throw: Wrap a function to catch and suppress errors.
    • wait-for: Wait until a condition is true, with polling and abort support.
    • clone: Shallow clone a value (non-recursive).
    • deep-clone: Deep clone a value using structuredClone.
    • deep-merge: Deep merge two objects, returning a new object.
    • deep-merge-inplace: Deep merge two objects, mutating the target.
    • defaults: Assign default options to an object (like lodash.defaults).
    • is-deep-equal: Deep equality check using strict strategy.
    • is-equal: Various equality comparison strategies for values.
    • camelCase: Convert a string to camelCase.
    • kebabCase: Convert a string to kebab-case.
    • lowerCase: Convert a string to lower case with spaces.
    • lowerFirst: Lowercase the first character of a string.
    • snakeCase: Convert a string to snake_case.
    • startCase: Convert a string to Start Case.
    • upperCase: Convert a string to UPPER CASE with spaces.
    • upperFirst: Uppercase the first character of a string.
    • capitalize: Capitalize the first letter and lowercase the rest of a string.
    • join-non-empty: Join non-empty strings with a separator.
    • prefix-lines: Prefix each line of a string with a given prefix.
    • replace-diacritics: Replace diacritics in a string with base characters.
    • truncate: Truncate a string to a given length, with options.
    • average: Functions for mean, geometricMean, harmonicMean, rootMeanSquare, median, mode, midrange.
    • minmax: Get the minimum and maximum from a list of numbers.
    • clamp: Clamp a number between a minimum and maximum value.
    • random: Get a random value between min and max.
    • round: Round a number to a specified number of decimal places or precision.
    • scale: Scale a number in 0..1 to a new range.
    • to-fixed-length: Format a number to a fixed-length string.
    • abortable-promise: Promise subclass with abort support.
    • regexp-escape: Escape special characters for use in RegExp patterns.
    • delay: Delay execution of a callback by a given number of milliseconds.
    • periodical-timer: Timer that calls a callback at regular intervals.
    • sleep: Sleep for a specified number of milliseconds.
    • timer: Simple timer that calls a callback after a delay.

    • exec-async: Run a shell command asynchronously and get the result.
    • node-exec-error: Error class for child process execution errors.
    • process-exit-error: Error class for process exit errors.

    • import-module: Dynamically import a module.
    • resolve-module: Resolve a module path like Node.js does.
    • get-random-ephemeral-port: Get a random available ephemeral port.

    Shell Utilities

    • parse-env-variable: Parse an environment variable string.
    • stringify-env-variable: Stringify an environment variable value.
    • escape-command: Escape a shell command for safe execution.

    Text & File Utilities

    • text-file: Read, write, and analyze text files with encoding and formatting options.
    • json-file: Read and write JSON files with comments and formatting support.
    • json5-file: Read and write JSON5 files.
    • jsonc-file: Read and write JSONC (JSON with comments) files.
    • ini-file: Read and write INI files.
    • toml-file: Read and write TOML files.
    • yaml-file: Read and write YAML files.

    Time

    • format-duration: Format a duration in milliseconds as a human-readable string.

    3rd Party Utilities

    • chardet-charset-to-buffer-encoding: Charset conversion helpers for chardet.
    • editorconfig-charset-to-buffer-encoding: Charset conversion helpers for editorconfig.

    Types
